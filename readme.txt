Adds fixed social media buttons to the sides of your wordpress website.

== Description ==

Let's you input links to social media. They'll be displayed on either the very right or very left side on all of your wordpress sites and scroll along, thus keeping social media links always in sight - but never intrusive. 
Supports (so far):

*   facebook
*   google+
*   tumblr
*   twitter
*   deviantart
*   youtube.

== Installation ==

1.  Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2.  Activate the plugin through the 'Plugins' menu in WordPress
3.  Place `&lt;?php do_action('wp_footer'); ?&gt;` in your templates
